CodeBeamer Admin Session Test

This page helps you test the admin session validation for CodeBeamer auto-login.

How it works:

  1. Admin (sejin.park) must be logged in to CodeBeamer first
  2. Use the admin's JWT token to validate the session
  3. If admin session is valid, vectorCAST can be auto-logged in
